# LineWatt Library Legal Framework

> **Draft for legal review.** Nothing in this directory is approved for publication, acceptance, or operational reliance until qualified counsel and the responsible business owners approve it.

## 1. Purpose

This directory is the governed Version 1.0 legal-documentation subsystem for LineWatt Library. It separates contracts, notices, policies, internal controls, and operational registers while using shared definitions and explicit precedence.

## 4. Publication and acceptance

Only an Approved document may be Published. Publication requires completed metadata, cleared placeholders, counsel review, an archived prior version, stable public URL, and acceptance controls described in the [Publication and Acceptance Matrix](./appendices/PUBLICATION_AND_ACCEPTANCE_MATRIX.md). Internal documents and registers must not be exposed merely because they are stored here.

## 5. Updating the framework

Follow the [Legal Style Guide](./LEGAL_STYLE_GUIDE.md) and [Legal Versioning](./LEGAL_VERSIONING.md); update affected cross-references, matrices, [Version History](./appendices/VERSION_HISTORY.md), and [Legal Framework Changelog](./appendices/CHANGELOG.md); run link and placeholder checks; and obtain the approvals required by the architecture.
