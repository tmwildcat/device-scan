# Legal Governance Security

Every Counsel route requires a named legal permission. Legal Counsel receives no entitlement to Private Datasets. Editing, approval, publication, sensitive evidence, and export permissions are distinct configuration entries.

Markdown rendering strips raw HTML and unsafe links. Public records use UUIDs. Published artefacts, acceptances, reviews, manifests, and audit events are immutable or append-only. Foreign keys restrict historical deletion. IP capture is configurable and should be masked outside the sensitive-evidence permission.

Production hardening still requires download policies, upload validation for executed artefacts, rate limits, cross-organisation policy tests, storage write-once controls, and a verified retention schedule.
